Proyecto IMC

Aplicación web que calcula el Índice de Masa Corporal (IMC) y guarda el historial de cada usuario.

Cómo usar

Conecta la base de datos MySQL.

Ejecuta el proyecto y abre http://localhost:8080
.

Qué hace

Permite registrarte e iniciar sesión.

Calcula tu IMC y guarda tus resultados.

Muestra tu historial de mediciones.
